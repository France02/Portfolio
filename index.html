About Me

Ciao, sono Francesco. Come Full Stack Developer, la mia missione è progettare e costruire applicazioni web complete end-to-end. La mia esperienza spazia dalla creazione di interfacce utente interattive con React e JavaScript, alla progettazione di robuste architetture back-end con Java, Spring Boot e database SQL.

Affronto ogni progetto con una mentalità orientata alla soluzione, cercando di bilanciare estetica e funzionalità per creare prodotti performanti e di alta qualità. Sono un sostenitore del codice pulito e della collaborazione in team per raggiungere obiettivi ambiziosi.

Le mie Tecnologie:

Java Spring Boot SQL & MySQL React JavaScript (ES6+) HTML5 & CSS3 Git & GitHub Figma Bootstrap
